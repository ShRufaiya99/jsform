# Dark Mode Toggle & Form Validation 🌗

This project uses a dynamic form validation system with Dark/Light mode switching feature.

## ✨ Features
- 🌗 **Theme Switcher:** `darkstyle.css` and `style.css` toggle for Dark/Light mode control.
- ✅ **Form Validation:** User input check without error message (script.js).
- 📱 **Responsive Design:** Mobile and Desktop devices are compatible.

## 📂 Project Structure
- `index.html` - Main structure.
- `style.css` - Light mode styling.
- `darkstyle.css` - Dark mode styling.
- `script.js` - Logic for validation and theme toggle.

## 🚀 How to Run
1. Clone the repository.
2. `index.html` file-ti browser-e open koro.
